# Analysis Plots

## November_Monitoring_Plots.jpg

### Overview
Comprehensive visualization of the monitoring period (November 14-15, 2025) from the Bachupally deployment. This composite plot shows time-series trends for all environmental parameters collected by the multi-sensor ESP32 node.

### Data Visualization

The plots illustrate:
- **Pollutant concentration patterns** - PM2.5, PM10, TVOC, eCO2, NO2
- **Diurnal (day-night) variations** - Industrial activity cycles and ambient conditions
- **Meteorological correlations** - Temperature and humidity trends
- **Industrial activity impact** - Elevated pollution during operational hours
- **Baseline data** - Representative data for pharmaceutical zone monitoring

### Parameters Displayed
- **PM10** - Particulate Matter (10 µm)
- **TVOC** - Total Volatile Organic Compounds
- **eCO2** - Equivalent Carbon Dioxide
- **Temperature** - Ambient temperature (°C)
- **Humidity** - Relative humidity (%)

### Analysis Period
**November 14-15, 2025** (One full day)
- High-resolution data capture
- 6-second local logging frequency
- 15-second cloud upload interval
- Continuous 24-hour operation

### Key Observations
- Clear diurnal patterns in pollutant concentrations
- Correlation between temperature/humidity and pollution levels
- Elevated readings during industrial operational hours
- Recovery periods during night hours
- Air quality metrics for industrial zone
